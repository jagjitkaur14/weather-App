# GitHub Update Steps

## 1. Add the README

1. Open: https://github.com/jagjitkaur14/weather-App
2. Click **Add file** → **Create new file**
3. File name: `README.md`
4. Paste the content from the updated README file.
5. Commit message: `Add portfolio-ready README for Nimbus Weather App`
6. Click **Commit changes**

## 2. Update repository About description

Use this description:

A responsive weather dashboard using OpenWeatherMap API with real-time weather, hourly forecast, 7-day forecast, unit toggle, saved default location, and local preference storage.
